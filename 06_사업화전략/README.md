## 규약

- 수치와 사실은 출처를 함께 답니다. 출처가 없으면 추정으로 표시하고 확정하지 않습니다.
- 약점은 숨기지 않습니다. 지식·시맨틱(L4)은 현재 자체 역량이 부족한 확보 과제로 둡니다.
- 확정 사실과 추정, 벤더 발표 수준 주장을 구분합니다.
