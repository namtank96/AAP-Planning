과실 산정

과실을 한 줄 규칙 추론으로 산정 — 규칙기반 비율과 판례(유사사례)가 어긋나도 충돌이 드러나지 않는다. 어떤 규칙이 마지막에 떴느냐로 답이 흔들림.

분쟁 시 "왜 이 과실?" 방어가 약하다
⚠② 교차검증 부재
합의금 · 사기

사기 점수는 앞단에서 뜨지만(78점), 합의금 산정과 독립적으로 맞물리지 않는다. 합의금 적정성을 도전·게이트하는 단계가 명시적이지 않아 권고에 그대로 포함.

한계 ① 근거·재현성 해결
✓ 독립 산출 + 충돌 표면화

과실·유사사례를 독립 에이전트가 각각 산출 → Reconciler가 규칙↔판례 불일치를 띄우고 양쪽 근거를 DAG에 보존. 동일입력=동일결과(재현성ID).

한계 ② 교차검증 해결
✓ 합의금 독립 게이트

사기·유사사례 에이전트가 합의금 적정성을 독립 도전 → 충돌 시 합의금분 ₩90만 보류·SIU. 부당지급을 명시적으로 차단.

한계 ③ 병목·추적 해결
✓ 차등 HITL + 전건 DAG

Gateway가 위험·금액별로 라우팅(대물 자동 / 과실 사정사 / 합의금 SIU). 사람은 분쟁·고액만, 전 과정은 감사 DAG로 보존.
